Kingdom Hearts

This is a simple website with basic information about the game Kingdom Hearts.

About the game

Kingdom Hearts is an action role-playing game series developed by Square Enix in collaboration with Disney. It combines original storytelling with characters from Disney and Final Fantasy.

Main characters

  • Sora — The main hero who wields the Keyblade.
  • Donald Duck — A royal magician of Disney Castle.
  • Goofy — Captain of the royal knights.
  • Riku — Sora’s best friend.
  • Kairi — A close friend connected to the light.
  • Mickey Mouse — A Keyblade Master.

Gameplay

The games feature real-time combat, magic abilities, and journeys through multiple Disney-inspired worlds.

Release info

The first Kingdom Hearts game was released in 2002 for the PlayStation 2.
