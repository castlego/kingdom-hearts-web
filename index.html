Kingdom Hearts

This is a simple website with basic information about the game Kingdom Hearts.

About the Game

Kingdom Hearts is an action role-playing game series made by Square Enix in collaboration with Disney. It combines original storytelling with Disney and Final Fantasy characters.

Main Characters

  • Sora – The main hero who wields the Keyblade.
  • Donald - A royal magician of disney castle.
  • Goofy - Captain of the royal knights.
  • Riku – Sora’s best friend.
  • Kairi – A close friend connected to the light.
  • Mickey Mouse – A Keyblade Master.

Gameplay

The game features real-time combat, magic abilities, and journeys to multiple Disney-inspired worlds.

Release Info

The first Kingdom Hearts game was released in 2002 for the PlayStation 2.
